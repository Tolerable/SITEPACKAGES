FAQ

Do I need coding skills? +
No. The setup wizard lets you configure everything visually. Colors, products, effects, terminology - all through dropdown menus and input fields. The wizard generates your config files automatically.
What's the difference between Starter and Pro? +
Starter is a static template - great for simple stores with products defined in config files. Pro adds user authentication, database integration (Supabase), shopping cart, and order management through Netlify serverless functions.
Can I sell digital products? +
Yes. Each product can be marked as physical or digital. Digital products support download URLs. The checkout flow handles them differently - no shipping address required.
What payment methods are supported? +
The template supports email-based checkout (customers email their order), Ko-fi integration, and external checkout URLs (PayPal, Stripe payment links). Direct payment processing can be added by customizing the checkout flow.
Is hosting included? +
No, but the template is designed for free hosting. Netlify's free tier handles static sites and serverless functions. Supabase has a generous free tier for database and auth. You can run a full store at zero hosting cost.
Can I customize the design beyond the wizard? +
Absolutely. The template uses clean HTML, CSS, and vanilla JavaScript. Developers can modify any file. The CSS uses variables for easy theming. All code is yours to edit.
What are the visual effects? +
The effects library includes: animated starfield backgrounds, particle systems, falling leaves, spotlight/cursor follow effects, floating custom images, and scroll animations. All configurable through the wizard - enable/disable each, set intensity.
Is there a demo? +
You're looking at it. This site (sitepackages.net) runs on the SITEPACKAGES template. The packages listed above are sold through the same system they help you build.
